# Task Management (Test Assignment)

This repository contains a test assignment: a task management system with collaboration and project management features, built with Angular (web) and NestJS (api).

## Requirements
- Docker

## Quick Start
- Run: `docker compose up --build`
- Web app: `http://localhost:4200`
- API: `http://localhost:3000`

## Structure
- `web/` Angular client
- `api/` NestJS backend
